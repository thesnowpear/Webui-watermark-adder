# 快速开始指南

## 安装步骤

### 1. 找到 WebUI Forge 的 extensions 目录

通常位于：
```
stable-diffusion-webui-forge/extensions/
```

### 2. 克隆或复制本项目

**方法 A：使用 Git**
```bash
cd stable-diffusion-webui-forge/extensions/
git clone <repository-url> sd-webui-watermark-adder
```

**方法 B：手动复制**
将整个项目文件夹复制到 `extensions` 目录下

### 3. 重启 WebUI Forge

关闭并重新启动 WebUI Forge

### 4. 找到水印添加标签页

在 WebUI 界面中找到 "Watermark Adder" 标签页

## 使用教程

### 基础使用流程

#### 步骤 1：上传图片
1. 在左侧"图像编辑区"点击上传图片
2. 或者从 txt2img/img2img 标签页点击"发送到水印区"按钮

#### 步骤 2：创建或选择水印

**创建文字水印：**
1. 切换到"文字水印"标签页
2. 输入水印文字（如 "© 2024"）
3. 调整字体大小、颜色、透明度
4. 输入水印名称
5. 点击"💾 保存文字水印"

**上传图片水印：**
1. 切换到"图片水印"标签页
2. 点击上传水印图片（建议使用 PNG 透明背景）
3. 点击"📤 上传图片水印"

#### 步骤 3：添加水印到图片
1. 在"已保存的水印"区域点击选择一个水印
2. 鼠标移动到图像编辑区，水印会跟随鼠标
3. 使用滚轮调整水印大小
4. 按住 Ctrl + 滚轮调整水印角度
5. 点击鼠标左键在该位置添加水印
6. 可以重复添加多个水印

#### 步骤 4：生成和保存
1. 点击"✨ 生成水印图片"按钮
2. 在右侧预览区查看效果
3. 选择保存方式：
   - **💾 保存图片**：保存为普通 PNG 图片
   - **📦 保存可解压图片包**：保存为特殊格式（见下文）

### 高级功能

#### 可解压图片包

这是一个特殊功能，生成的文件：
- 看起来是普通的 PNG 图片（显示带水印的版本）
- 但可以解压出原始图片

**如何使用：**
1. 点击"📦 保存可解压图片包"
2. 找到保存的文件（如 `extractable_1234567890.png`）
3. 将文件后缀改为 `.zip`：
   ```bash
   # Windows
   ren extractable_1234567890.png extractable_1234567890.zip

   # Linux/Mac
   mv extractable_1234567890.png extractable_1234567890.zip
   ```
4. 用解压软件打开，可以看到 `original_image.png`（原始图片）

**应用场景：**
- 公开分享带水印的图片
- 同时保留原图用于后续编辑
- 一个文件包含两个版本

#### 快捷键和技巧

- **滚轮**：快速调整水印大小
- **Ctrl + 滚轮**：快速调整水印角度
- **↶ 撤销**：删除最后添加的水印
- **🔄 刷新**：重新加载水印库

#### 水印管理

**删除水印：**
1. 在"已保存的水印"区域选中要删除的水印
2. 点击"🗑️ 删除选中"按钮

**查看所有水印：**
点击"🔄 刷新"按钮更新水印库

## 常见问题

### Q: 为什么看不到 Watermark Adder 标签页？
A: 确保：
1. 项目放在正确的 `extensions` 目录下
2. 已经重启 WebUI Forge
3. 检查控制台是否有错误信息

### Q: 水印跟随鼠标不流畅？
A: 这可能是因为：
1. 图片太大，尝试缩小图片尺寸
2. 浏览器性能问题，尝试关闭其他标签页

### Q: 文字水印显示为方块？
A: 系统缺少对应字体，会使用默认字体。可以：
1. 安装更多字体
2. 使用图片水印代替

### Q: 可解压图片包无法解压？
A: 确保：
1. 文件后缀已改为 `.zip`
2. 使用支持 ZIP 格式的解压软件（WinRAR、7-Zip、unzip 等）

### Q: 保存的图片在哪里？
A: 默认保存在：
```
stable-diffusion-webui-forge/outputs/watermarked/
```

## 技术支持

2. 检查 `watermarks` 目录权限
3. 确认 Pillow 库已正确安装
4. 提交 Issue 到项目仓库
