# 快速开始

## 安装

### 1. 找到 WebUI Forge 的 extensions 目录

```
stable-diffusion-webui-forge/extensions/
```

### 2. 克隆或复制项目

**方法 A：使用 Git**
```bash
cd stable-diffusion-webui-forge/extensions/
git clone <repository-url> sd-webui-watermark-adder
```

**方法 B：手动复制**

将整个项目文件夹复制到 `extensions` 目录。

### 3. 重启 WebUI Forge

关闭并重启 WebUI Forge，在顶部标签栏找到 **Watermark Adder**。

## 界面布局

界面分为三栏：

| 左栏：水印库 | 中栏：编辑区 | 右栏：预览与保存 |
|:---|:---|:---|
| 图片/文字/形状水印管理 | 图像编辑与水印放置 | 生成预览与保存 |
| 上传、删除、选择 | 缩放、平移、快捷键 | 普通保存 / 可解压包 |

## 使用教程

### 第一步：上传图片

- 在编辑区拖拽或点击上传图片
- 或点击 **获取最新图片**（自动扫描 `outputs/` 获取最新图，排除 `outputs/watermarked/`）

### 第二步：创建或选择水印

**创建文字水印：**
1. 切换到 “文字水印” 标签
2. 输入水印文字
3. 调整字号和颜色
4. 输入水印名称并点击 “保存”

**上传图片水印：**
1. 切换到 “图片水印” 标签
2. 拖拽或点击上传水印图片（推荐 PNG 透明背景）
3. 上传后自动保存到水印库

**创建形状水印：**
1. 切换到 “形状水印” 标签
2. 选择形状与效果模式：`颜色` / `模糊` / `马赛克`
3. 调整羽化和效果参数

### 第三步：添加水印到图片

1. 在水印库中点击选择一个水印
2. 鼠标移入编辑区预览
3. 文字/图片水印：**左键点击** 放置
4. 形状水印：**左键拖拽** 绘制
5. 可重复添加多个水印

### 第四步：编辑已有水印

1. 点击画布中的水印以选中
2. 拖拽水印主体移动
3. 拖拽控制点缩放
4. 缩放时按住 **Shift** 可切换比例锁定

### 第五步：浏览图像

- **滚轮**：缩放图像
- **左键拖拽**：平移图像
- **空格 + 左键拖拽**：平移图像
- **双击**：重置缩放与平移

### 第六步：生成与保存

1. 点击 **生成水印图片**（默认保存到 `outputs/watermarked/`）
2. 在右侧预览结果
3. 选择保存方式：
   - **保存图片**：标准 PNG
   - **保存可解压包**：特殊可解压格式

## 可解压水印包

生成的文件：
- 看起来是普通 PNG（水印版本）
- 将后缀 `.png` 改成 `.zip` 即可解压出原图

**解压示例：**
```bash
# Windows
ren extractable_1234567890.png extractable_1234567890.zip

# Linux/Mac
mv extractable_1234567890.png extractable_1234567890.zip

# 解压
unzip extractable_1234567890.zip
# 输出: original_image.png
```

| 操作 | 快捷键 |

| 缩放图像 | 滚轮 |

## 常见问题

### Q: 为什么看不到 Watermark Adder 标签？
确认项目放在 `extensions` 目录并重启 WebUI Forge，检查控制台是否有报错。

### Q: 大图加载黑屏？
扩展会自动重试等待图片解码完成（最多约 9 秒）。如果图片很大，请耐心等待或先缩小再上传。

### Q: 保存的图片在哪里？
默认保存到 WebUI 根目录下的 `outputs/watermarked/`。

### Q: 水印文字显示为方块？
系统缺少字体时会自动回退默认字体，建议使用图片水印。
