# 项目需求文档

## 项目概述

**项目名称**: WebUI-Forge 水印添加扩展
**项目类型**: Stable Diffusion WebUI Forge 可安装扩展程序
**开发目的**: 为 AI 绘画工具 webui-forge 提供专业的图片水印添加功能

## 重要说明

⚠️ **这是一个 webui-forge 扩展程序，不是独立应用**

本项目必须安装在 Stable Diffusion WebUI Forge 的 `extensions` 目录下才能运行。

### 安装位置示例
```
stable-diffusion-webui-forge/
├── extensions/
│   └── sd-webui-watermark-adder/  ← 本项目应该放在这里
│       ├── scripts/
│       ├── javascript/
│       └── ...
├── models/
├── outputs/
└── webui.py
```

## 核心功能需求

### 1. UI 布局

#### 左侧编辑区域
- **图像编辑区**（左上）
  - 上传或接收图片
  - 可视化添加水印
  - 实时预览水印位置

- **编辑控件**
  - 撤销按钮：撤销最后添加的水印
  - 清除图片按钮：清空编辑区
  - 水印大小滑块：10-500px
  - 水印角度滑块：0-360度

- **水印选择区**（左下）
  - 标签页 1：文字水印
    - 输入文字内容
    - 设置字体大小
    - 选择颜色
    - 调整透明度
    - 保存到本地
  - 标签页 2：图片水印
    - 上传图片文件
    - 自动保存到 watermarks/images/

- **水印库**
  - 显示所有已保存的水印
  - 点击选择水印
  - 删除不需要的水印

- **生成按钮**
  - 生成最终带水印的图片

#### 右侧预览和保存区域
- **预览区**
  - 显示添加水印后的效果

- **保存选项**
  - 保存图片：普通 PNG 格式
  - 保存可解压图片包：Polyglot 文件（外观是图片，改后缀为 .zip 可解压出原图）

### 2. 交互逻辑

#### 鼠标交互
- **鼠标移动**：选中水印后，水印跟随鼠标在编辑区移动
- **左键点击**：在点击位置添加水印
- **滚轮**：调整水印大小
- **Ctrl + 滚轮**：调整水印旋转角度

#### 工作流程
1. 用户上传图片到编辑区
2. 在水印选择区创建或选择水印
3. 鼠标移动到编辑区，水印跟随鼠标
4. 点击添加水印到图片上
5. 可以添加多个水印
6. 点击生成按钮
7. 在预览区查看效果
8. 选择保存方式

### 3. 额外功能

#### 与 WebUI 集成
在 webui-forge 原有的图像预览区下方添加"发送到水印区"按钮：
- 点击后跳转到水印添加标签页
- 自动将预览的图片加载到编辑区
- 方便用户直接为生成的图片添加水印

## 技术实现

### 后端（Python）
- **框架**: Gradio（webui-forge 使用的 UI 框架）
- **图像处理**: Pillow (PIL)
- **文件管理**: json, pathlib
- **Polyglot 文件**: zipfile

### 前端（JavaScript）
- **Canvas API**: 实现水印的可视化编辑
- **事件处理**: 鼠标交互、滚轮控制

### 数据存储
- **文字水印**: JSON 文件存储在 `watermarks/texts/`
- **图片水印**: 图片文件存储在 `watermarks/images/`
- **输出文件**: 保存在 `outputs/watermarked/`

## 特殊技术：Polyglot 文件

### 什么是 Polyglot 文件？
一个文件同时符合多种文件格式的规范。在本项目中，创建的文件：
- 作为 PNG 图片打开：显示带水印的图片
- 改后缀为 .zip 打开：可以解压出原始图片

### 实现原理
1. 将带水印的图片数据写入文件
2. 将包含原图的 ZIP 数据追加到文件末尾
3. PNG 格式允许在文件末尾有额外数据，不影响图片显示
4. ZIP 工具可以识别文件中的 ZIP 数据并解压

### 用户使用方法
```bash
# 1. 保存可解压图片包，得到 extractable_xxx.png
# 2. 将文件改名
mv extractable_xxx.png extractable_xxx.zip
# 3. 解压
unzip extractable_xxx.zip
# 4. 得到 original_image.png（原始图片）
```

## 注意事项

1. **依赖管理**: 确保 Pillow 版本 >= 9.0.0
2. **字体支持**: 不同系统的默认字体不同，需要处理字体缺失情况
3. **文件权限**: 确保 watermarks 和 outputs 目录有写入权限
4. **性能优化**: 大图片处理时注意内存使用
5. **跨域问题**: Canvas 操作时注意图片跨域限制
