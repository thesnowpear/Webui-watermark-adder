# WebUI-Forge 水印添加扩展

[English](README.md)

这是一个用于 Stable Diffusion WebUI Forge 的水印扩展，提供可视化编辑、实时预览以及可解压水印包功能。

## 功能

- **文字水印**：创建并保存自定义文字水印，支持颜色、大小、透明度
- **图片水印**：上传并管理图片水印（推荐 PNG 透明背景）
- **形状水印**：正方形、长方形、圆形、椭圆，支持三选一效果模式：`颜色` / `模糊` / `马赛克`
- **羽化**：形状水印边缘柔化
- **可视化编辑**：Canvas 叠加层，鼠标跟随预览，文字/图片点击放置，形状拖拽绘制
- **选中并编辑已有水印**：在画布中移动、缩放、旋转或删除
- **缩放与平移**：滚轮缩放，拖拽平移，双击重置
- **快捷键控制**：
  - 滚轮：缩放图像
  - 左键拖拽：平移图像
  - 空格 + 左键拖拽：平移图像
  - 双击：重置视图
  - Ctrl + 滚轮：调整水印大小（±20px）
  - Shift + 滚轮：调整旋转角度
  - Alt + 滚轮：调整透明度
- **撤销 / 清空**：撤销最后一个水印或清空全部
- **两种保存模式**：
  - 普通保存：标准 PNG 水印图
  - 可解压水印包：可改名为 `.zip` 提取原图
- **自动保存**：生成时默认保存到 `outputs/watermarked/`
- **获取最新图片**：自动扫描 `outputs/` 获取最新图（排除 `outputs/watermarked/`）
- **可空水印生成**：不添加水印也允许生成，输出原图副本

## 安装

1. 进入 WebUI Forge 的 `extensions` 目录
2. 克隆仓库：
   ```bash
   git clone https://github.com/yourusername/sd-webui-watermark-adder.git
   ```
3. 重启 WebUI Forge

## 使用

1. 打开 **Watermark Adder** 选项卡（与 txt2img/img2img 同级）
2. 上传图片，或点击 **获取最新图片** 从 `outputs/` 加载
3. 在水印库中创建或选择水印
4. 鼠标移入编辑区预览
5. 点击放置文字/图片水印
6. 拖拽绘制形状水印
7. 点击已有水印进入编辑，拖拽控制点缩放或旋转
8. 点击 **生成** 输出结果（默认保存到 `outputs/watermarked/`）

## 编辑提示

- 缩放时按住 **Shift** 可切换比例锁定
- 形状 **模糊** 控制模糊强度
- 形状 **马赛克** 控制每个方块像素尺寸
- 缩放或拖拽期间会自动切换为轻量预览，停手后恢复高清

## 技术栈

- Python 3.x + Gradio
- Pillow (PIL)
- JavaScript + HTML5 Canvas
- Polyglot 文件（PNG + ZIP）
